## 🎯 Demo en Vivo

**👉 [Ver Demo](https://useche70.github.io/matrix/)**

Una recreación fiel del icónico efecto de lluvia digital de la película **The Matrix** usando HTML5 Canvas, CSS3 y JavaScript vanilla.

## ✨ Características

- 🌧️ **Lluvia de caracteres** japoneses auténticos como en la película
- ⚡ **Animación fluida** optimizada con `requestAnimationFrame`
- 🎨 **Múltiples colores** disponibles (verde Matrix, rojo, azul, etc.)
- 🎛️ **Controles interactivos** en tiempo real
- 📱 **Responsive design** que se adapta a cualquier pantalla
- ⌨️ **Controles de teclado** para una experiencia completa
- 🖱️ **Efectos especiales** al hacer clic
- 🌟 **Caracteres brillantes** aleatorios para mayor realismo

## 🎮 Controles

### 🖱️ Controles en Pantalla
- **⏸️ Pausar/Reanudar** - Para pausar o continuar la animación
- **⚡ Cambiar Velocidad** - Cicla entre diferentes velocidades
- **🎨 Cambiar Color** - Cambia entre diferentes colores
- **🎚️ Deslizadores** - Ajusta densidad y velocidad en tiempo real

### ⌨️ Controles de Teclado
- **Espacio**: Pausar/Reanudar animación
- **⬆️ Flecha Arriba**: Aumentar velocidad
- **⬇️ Flecha Abajo**: Disminuir velocidad
- **C**: Cambiar color
- **F11**: Pantalla completa (recomendado)

### 🖱️ Efectos Especiales
- **Clic en pantalla**: Crear explosión de caracteres en esa posición
- **Caracteres brillantes**: Aparecen aleatoriamente para mayor realismo

## 🚀 Instalación Local

1. **Clona el repositorio**
   ```bash
   git clone https://github.com/useche70/matrix.git
   cd matrix
   ```

2. **Abre en navegador**
   - Doble clic en `index.html`
   - O ejecuta `Ejecutar_Matrix.bat` (Windows)

3. **¡Disfruta del efecto Matrix!**
   - Presiona F11 para pantalla completa
   - Experimenta con los controles

## 🎨 Personalización

El código está bien comentado y es fácil de modificar:

- **Colores**: Modifica el array `colors` en el JavaScript
- **Caracteres**: Cambia la variable `matrixChars` 
- **Velocidad**: Ajusta `animationSpeed`
- **Densidad**: Modifica `fontSize` y `columns`
